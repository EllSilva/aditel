GPS Rastreador Automóvel

Click aqui para Ver Descricao

Descrição

Desenhado para funcionar com base na rede GSM/GPRS (850/900/1800/1900MHz ) e satélites GPS, este produto está habilitado para efetuar Rastreamento em tempo real, localizar, bloquear, acesso ao sistema áudio, monitorar qualquer alvo remoto por SMS ou via Plataforma de Gestão Profissional com acesso à internet. Adicionalmente, quando instalado por uma equipa de verdadeiros profissionais, o equipamento estará habilitado para fornecer ao usuário o sistema de alarme sonoro básico anti-choque para melhor proteção da sua viatura.
