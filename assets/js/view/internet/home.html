Antena Sectorial robusta BS para uso Externo sem fio 5 GHz com 802.11ac, tecnologia para cobertura sem fio de ponta ou conexão PtMP com a tecnologia 11ac. A tecnologia MIMO 2*2 integrada de 867Mbps 5GHz torna-o 3x mais rápido do que o padrão 802.11n. A potência tx é aumentada até 26dBm para uma distância incrível de longa distância. Estação base Embutido Antena Sectorial Integrada dual-pola de até 19dBi, a estação base 11ac é fácil de fornecer longo alcance para a cobertura externa de CPEs, smart phone e laptops.
